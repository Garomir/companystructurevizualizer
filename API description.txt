Запускаем проект, запускаем Postman, выбираем тип запроса (POST,GET,PUT,DELETE) и пишем путь api-запроса, например:

Получение всех отделов:
GET	http://localhost:8080/company/department

Получение отдела по id:
GET	http://localhost:8080/company/department/{id}

Получение всех работников:
GET	http://localhost:8080/company/worker

Получение всех работников из одного отдела по id отдела:
GET	http://localhost:8080/company/department/worker/{id}

Получение работника по id:
GET	http://localhost:8080/company/worker/{id}

Удаление отдела по id:
DELETE	http://localhost:8080/company/department/{id}

Удаление работников по id:
DELETE	http://localhost:8080/company/worker/{id}

Добавление отдела:
POST	http://localhost:8080/company/department
дополнительно надо во вкладке Body (raw-Json) написать Json, например:
{
	"name":"Отдел кадров"

}

Добавление работника в отдел по id отдела:
POST	http://localhost:8080/company/department/{id}/worker
дополнительно надо во вкладке Body (raw-Json) написать Json, например:
{
	"firstname":"Гуля",
	"lastname":"Гулевна",
	"position":"кадровик"
}

Изменение отдела по id:
PUT	http://localhost:8080/company/department/{id}
дополнительно надо во вкладке Body (raw-Json) написать Json с новой инфой отдела, например:
{
	"name":"Бухгалтерия"

}

Изменение работника по id:
PUT	http://localhost:8080/company/worker/{id}
дополнительно надо во вкладке Body (raw-Json) написать Json с новой инфой работника, например:
{
	"firstname":"Вася",
	"lastname":"Васильев",
	"position":"программист"
}
